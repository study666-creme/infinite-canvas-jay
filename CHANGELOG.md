# Changelog

## Unreleased

### Added

- 3D 导演台：Agent 可载入分镜包、自动恢复角色/道具/机位，并把单镜头或批量预演截图写回画布。
- 结构化项目黑板与全链路创作总监 Agent。
- 本地创作资料库：增量索引、知识卡二次审核、故事案例库、字幕提取与静态检索包。
- 卡藏账号登录、模型价格读取和 Prompt Hub 图片生成联动。

### Improved

- 桌面与手机画布布局、节点提示词面板、Agent 反馈和本地媒体持久化。
- Codex/Claude 本机 Agent 的工作区路由、会话恢复和画布 MCP 工具。
- 图片与视频生成的取消、失败节点清理、生成记录和首屏加载。

### Fixed

- 登录状态重复校验与错误 API 地址回退。
- 空格平移结束后画布继续跟随鼠标。
- 删除生成节点后异步结果重新创建节点。
- Prompt Hub 参考图上传、连接图片优先级和嵌套错误信息解析。
