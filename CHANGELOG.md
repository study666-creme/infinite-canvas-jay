# Changelog

## Unreleased

### Added

- 3D 导演台：Agent 可载入分镜包、自动恢复角色/道具/机位，并把单镜头或批量预演截图写回画布。
- 可选结构化项目状态与通用画布 Agent 工具链。
- 可选本地资料工具：增量索引、隔离二审、案例结构提取与字幕质量检查。
- 卡藏账号登录、实时模型目录、积分计价和图片/视频/文本生成联动。

### Improved

- 模型下拉在长名称场景下固定保留价格区域，名称超长时省略显示，价格不再溢出容器。
- 按住空格时高级光标会立即切换为手形，按下拖动时给出明确的抓取反馈，并支持从节点上开始平移。

- 桌面与手机画布布局、节点提示词面板、Agent 反馈和本地媒体持久化。
- Codex/Claude 本机 Agent 的工作区路由、会话恢复和画布 MCP 工具。
- 图片与视频生成的取消、失败节点清理、生成记录和首屏加载。
- 模型按文本、图片、视频和音频能力分组；可见模型由用户选择，新模型不会自动加入已有可选项。
- 图片张数、比例、分辨率、参考图上限和积分报价跟随实时模型目录，价格同时展示积分与人民币。
- 公开 Agent 提示收敛为通用工具协议，不再打包私有知识、短剧流程或个人预设。

### Fixed

- 旧配置升级后会移除未填写密钥的默认渠道，默认与附加选项统一使用卡藏 API 的分类型模型。
- 图片节点提示词编辑器仅接收纯文本；粘贴图片或富文本时不会把图片嵌入输入框。

- 登录状态重复校验与错误 API 地址回退。
- 空格平移结束后画布继续跟随鼠标。
- 删除生成节点后异步结果重新创建节点。
- Prompt Hub 参考图上传、连接图片优先级和嵌套错误信息解析。
- 未配置的默认渠道与跨模态幽灵模型出现在选择器中。
