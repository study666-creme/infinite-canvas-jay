# 节点与生成流程

## 文本节点

- 保存提示词、故事、剧本、分镜和说明。
- 双击内容直接编辑；节点工具栏可调整字号。
- 下方输入框可生成或改写文本。
- 作为上游连接到图片、视频、音频或配置节点时，内容会进入对应 prompt。

## 图片节点

- 支持上传、拖入、替换、下载、裁剪、蒙版编辑、多角度处理和反推提示词。
- 空节点可直接生成；已有图片可作为参考图继续生成。
- 批量结果保留为图片节点组，可展开、折叠和选择主图。
- 可保存到本地素材库或 Prompt Hub 卡片库。
- 图片自身的 `metadata.prompt` 不会自动变成视频 prompt；需要文字约束时请连接文本节点或写入下方输入框。

## 视频节点

- 支持上传、拖入、异步生成、播放、下载和重试。
- 可选取任意帧生成相连图片节点。
- 上游文本作为 prompt，上游图片/视频/音频分别作为参考媒体。
- 支持 OpenAI 风格、Seedance Agent Plan 和 xAI/Grok Imagine 任务。

远程视频能否播放取决于上游 URL、CORS、媒体格式和 `/v1/media/fetch` 网关，详见[视频生成与播放](/docs/overview/video-playback)。

## 音频节点

- 支持上传、拖入、播放和生成。
- 可配置音色、格式、语速与额外指令。
- 连接到视频生成流程时作为声音参考，具体支持程度由上游模型决定。

## 生成配置节点

生成配置节点统一读取上游内容，并保存当前节点自己的：

- 生成模式：文本、图片、视频或音频。
- 模型、比例、质量、数量和输入顺序。
- 视频时长、清晰度、声音与水印。
- 音频音色、格式和语速。

它适合把可复用流程固定在画布，而不是每次重新填写参数。

## 连线和 `@` 引用

- 左右连接点建立上下游关系。
- 节点下方输入框的 `@` 选择器可精确引用画布资源。
- 输入预览会显示最终发送的文本与参考媒体顺序。
- 删除生成中节点会取消关联请求，防止晚到结果重新创建已删除节点。

## 画布 Agent

Agent 会读取真实节点 ID 后再调用工具。创作任务优先维护项目黑板，并把故事、资产、分镜、审核和生成结果写入可见节点。不要向 Agent 手工复制内部 JSON。

## 3D 导演台

工具栏的“3D 导演台”可以导入分镜包、手动摆位和截图。Agent 也可以在文字分镜确认后自动载入最多 10 个镜头，并把预演图写回画布。

详细说明见[3D 导演台](/docs/overview/director-stage)。
