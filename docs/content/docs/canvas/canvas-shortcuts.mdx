# 鼠标、键盘与触控

## 桌面视图

- 空白处拖动：框选节点。
- `空格` + 拖动：平移画布；松开空格、鼠标、窗口失焦或指针取消都会结束平移。
- 滚轮或缩放控件：缩放画布。
- 小地图：快速定位节点区域。

## 选择与编组

- `Shift` + 点击/框选：追加或取消选择。
- `Ctrl / Cmd` + `A`：全选节点。
- 点击组内节点：选择并拖动整组。
- `Ctrl / Cmd` + 点击组内节点：仅选择单个成员。
- `Esc`：取消选择并关闭当前浮层。

## 编辑

- `Ctrl / Cmd` + `C`：复制选中节点和内部连线。
- `Ctrl / Cmd` + `V`：粘贴节点，或粘贴剪贴板文本/图片。
- `Delete / Backspace`：删除节点或连线。
- `Ctrl / Cmd` + `Z`：撤销。
- `Ctrl / Cmd` + `Shift` + `Z` 或 `Ctrl / Cmd` + `Y`：重做。
- 右键：打开画布、节点或连线上下文菜单。

## 移动端

- 空白画布单指拖动：平移。
- 双指捏合：围绕手势中点缩放。
- 点击节点：选择并打开对应工具。
- Agent 使用覆盖式面板，资产面板使用可用屏幕宽度。

手机适合检查画布、发送 Agent 任务和进行基础编辑。精细连线、大批量节点编排和复杂参数调整仍建议使用桌面端。

撤销历史包含节点、连线、视口、背景和助手会话变化；账号状态、全局模型配置和素材库操作不属于当前画布历史。
