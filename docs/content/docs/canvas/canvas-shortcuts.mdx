# 画布操作说明

本文档记录画布里常用的鼠标和键盘操作。画布右上角键盘图标或左下角帮助按钮也可打开同样的操作面板。

## 视图

- 拖动空白处：框选多个节点。
- `空格` + 拖动：平移画布。
- 鼠标滚轮：缩放画布。
- 缩放滑杆：精确调整缩放。
- 重置视图按钮：回到默认缩放和居中位置。

## 选择

- `Shift` + 拖动 / 点击：追加或取消选择节点。
- `Ctrl / Cmd` + `A`：全选画布节点。
- 点击组内节点：选中整组并一起拖动。
- `Ctrl / Cmd` + 点击组内节点：仅选中该节点，不选中整组。
- `Esc`：取消选择，并关闭当前浮层。

## 节点

- 拖动节点：移动节点位置。
- 点击视频节点：播放 / 暂停（拖动仍为移动节点）。
- 右键：打开上下文菜单。
- 右键 → 成组：将多个选中节点合并为一组。
- 右键 → 取消成组：解散当前节点组。

## 编辑

- `Ctrl / Cmd` + `C`：复制选中节点。
- `Ctrl / Cmd` + `V`：粘贴节点，或粘贴剪切板中的文本 / 图片。
- `Delete / Backspace`：删除选中的节点或连线。
- `Ctrl / Cmd` + `Z`：撤销。
- `Ctrl / Cmd` + `Shift` + `Z`：重做。
- `Ctrl / Cmd` + `Y`：重做。

## 素材

- 拖入图片 / 视频 / 音频文件：上传到画布。
- 导入图片按钮：从本地选择图片。
- 我的素材：选择素材后插入画布。

## 撤销和重做范围

撤销和重做会记录画布节点、连线、视口、背景模式和助手会话变化。画布项目名称、账号状态、全局 AI 配置和“我的素材”保存操作不属于当前画布历史。
