# AGPL 与第三方许可

本仓库基于 [basketikun/infinite-canvas](https://github.com/basketikun/infinite-canvas) 继续开发，主许可证为 [GNU AGPL-3.0](https://github.com/study666-creme/infinite-canvas-jay/blob/main/LICENSE)。

AGPL 允许学习、修改、部署、收费托管和商业运营。分发或通过网络提供修改版本时，需要按许可证向对应用户提供源码，并保留许可证和版权通知。

产品可以使用“卡藏提示词画布”品牌，但不能通过改名消除上游来源和开源义务。复杂的闭源交付、企业内网、插件边界和混合授权问题应咨询专业法律人士。

Three.js 使用 MIT 许可证。3D 模型、HDRI、贴图、字体、图片、视频、音频、书籍、字幕和案例拥有各自授权条件，必须逐项确认。

完整说明见仓库根目录 [OPEN-SOURCE.md](https://github.com/study666-creme/infinite-canvas-jay/blob/main/OPEN-SOURCE.md)。
