# 安全报告

不要在公开 Issue 中提交利用细节、API Key、Agent token、Cookie、私人画布、用户资料或未打码截图。

优先使用 [GitHub Private Vulnerability Reporting](https://github.com/study666-creme/infinite-canvas-jay/security/advisories/new)。入口不可用时，按仓库 [SECURITY.md](https://github.com/study666-creme/infinite-canvas-jay/blob/main/SECURITY.md) 中的联系方式私密提交。

重点范围包括浏览器存储跨账号读取、卡藏鉴权、WebDAV/媒体代理 SSRF、本机 Agent 未授权访问、工作区越界和密钥泄漏。
