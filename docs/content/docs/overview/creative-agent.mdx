# 创作总监 Agent

画布内置 Agent 已增强为「全链路创作总监」工作方式。它不是只列知识卡，而是在用户提出故事、剧本、分镜、导演、视觉、AI 视频、标题封面、网感和发布相关任务时，自动注入一套创作知识核心，并在输出前做二次质量审核。

## 知识质量门槛

内置知识只采用高可信来源的原创总结，不收录原书原文、完整视频字幕或逐字资料库：

- 故事/剧作：亚里士多德《诗学》、Lajos Egri、John Truby、Robert McKee、John Yorke、Syd Field、Linda Seger、Blake Snyder 等长期使用的剧作方法。
- 人物/心理/叙事认知：K.M. Weiland、Lisa Cron、Will Storr、David Mamet 等关于人物弧、动机、场景行动和叙事认知的原则。
- 导演/表演/视听：Judith Weston、Mick Hurbis-Cherrier、谢飞、王心语、韩小磊，以及摄影、分镜、色彩、剪辑、声音相关经典教材。
- 网文/短剧：用户指定 UP 主「天命斩水」B 站「写作教学」合集作为网文读者预期、情绪、爽点、结构和悬念方向参考，只抽取方法，不复述字幕。
- AI 视频生成：采用 OpenAI Sora、Google Veo、Runway 等官方文档中稳定的跨模型原则，如主体、动作、镜头、参考图、相机运动、光线色彩、声音和时序变化。
- 平台传播：参考 TikTok Creative Center、YouTube Creator Academy / Help、Meta Reels 创作建议等官方平台资料中稳定的留存、标题、封面、节奏和互动原则。

## 全链路覆盖

Agent 会在创作时按层判断当前缺口，而不是机械堆术语：

- 故事层：题材、受众、IP 母题、类型承诺、人物欲望、关系张力、结构、爽点、悬念、主题、情绪曲线。
- 剧本层：大纲、分场目的、场景价值转折、台词潜台词、动作线、反派压强、集尾钩子。
- 导演层：导演阐述、场面调度、表演行动动词、镜头景别、机位、运动、空间关系、节奏。
- 视听层：构图、光线、色彩、服装、场景、美术、道具、剪辑点、声音设计、音乐、静音。
- AI 生成层：角色图、场景图、参考图、首帧/尾帧、视频提示词、镜头拆分、运动约束、一致性、失败诊断、重抽策略。
- 发布质检层：前三秒、封面标题、小屏可读性、角色连续性、平台风险、版权/肖像/敏感风险。

## 审美和网感

网感不是简单追热词。Agent 会把专业创作翻译成平台用户能感知的冲突、情绪、身份和谈资：

- 钩子三测：3 秒能否看懂处境，10 秒是否有情绪回报，结尾是否留下继续观看或评论理由。
- 标题封面一体化：标题制造问题，封面让问题可视化，避免堆字和剧情摘要。
- 评论区触发点：预留可讨论的判断题、选择题或身份站队点。
- 审美去土味：避免廉价滤镜、尴尬金句、AI 味字幕和空泛电影感，优先保持清晰主体、真实材质、准确光源和统一视觉常量。
- 平台适配：抖音/快手重即时情绪和冲突，小红书重审美与可收藏，B 站重观点密度和人格信任，TikTok/Reels/Shorts 重无声可读和视觉动作。
- 趋势雷达：拆解趋势背后的主情绪、主身份、主冲突、主视觉符号、声音节奏和评论区争议，不把单次热梗当长期规律。

## 使用边界

这不是模型权重后训练，而是系统上下文层面的创作知识注入。Agent 不能声称拥有完整原书、完整字幕或实时热榜。用户提供书摘、字幕、账号样本或近期爆款案例后，Agent 可以把这些内容与内置知识一起用于更贴近账号的创作判断。

当用户只要结果时，Agent 应直接输出高质量结果；当存在明显质量风险时，只用一两句指出关键风险和改法。
