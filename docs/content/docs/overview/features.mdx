# 功能总览

## 账号与本地数据

- 用户区沿用卡藏账号登录，注册入口位于 `prompt-hubs.com`。
- 画布、素材和生成记录保存在浏览器 IndexedDB，并按卡藏用户 ID 分区。
- 首次登录会把旧的未分区本地数据复制到当前账号分区，避免直接丢失。
- 登录不是云同步。跨设备需要导出或配置 WebDAV。

## 无限画布

- 多画布项目的新建、重命名、删除、导入和导出。
- 画布平移、缩放、框选、多选、节点成组、撤销重做和小地图。
- 点阵、网格线、空白背景，以及浅色/深色主题。
- 节点左右连接点、上下游高亮和批量布局。
- 桌面鼠标操作与手机单指平移、双指缩放。

## 节点类型

| 节点 | 作用 |
| --- | --- |
| 文本 | 提示词、剧本、分镜、说明和 AI 文本结果 |
| 图片 | 上传、生成、编辑、裁剪、多角度处理和参考图 |
| 视频 | 上传、异步生成、播放、下载和选帧 |
| 音频 | 上传、语音/音频生成和视频声音参考 |
| 生成配置 | 汇总上游节点，配置文本、图片、视频或音频生成 |

图片批量结果支持主图、子图、展开与折叠。媒体节点使用 `storageKey` 读取 IndexedDB 或已授权本地目录中的 Blob，不把大体积 base64 长期写入画布 JSON。

## AI 生成

- OpenAI 兼容文本、图片、视频和音频接口。
- Gemini 文本与图片调用格式。
- OpenAI 风格异步视频、Seedance Agent Plan 与 xAI/Grok Imagine 视频任务。
- 图片/视频/音频引用、`@` 资源选择、模型、比例、数量、清晰度、时长、声音与水印参数。
- 生成进度、取消、失败重试和已删除节点的晚到结果拦截。

卡藏图片模型通过登录账号调用。自定义渠道通常由浏览器直接请求用户配置的服务。

## Prompt Hub 联动

- 从卡片库读取图片和提示词并插入画布。
- 把画布图片保存为 Prompt Hub 卡片。
- 参考图在提交卡藏生成前压缩副本并上传为受控媒体引用，原画布图片不被替换。
- 画布生成成功后写入共享图片生成记录。
- 读取卡藏文本、图片和视频模型、积分与实时价格。
- 模型按能力进入对应选择器；用户可以在设置中决定哪些模型显示，新模型不会自动改变已有选择。
- 图片张数、比例、分辨率、参考图上限和报价由当前模型目录决定，价格同时显示积分与人民币。
- 普通用户只看到公开模型名称；真实服务映射和请求日志仅供管理员排障。

## 画布 Agent

网页模型、本机 Codex、Claude 和无界面 API 可以作为画布 Agent 的执行后端，并使用同一套画布工具与可选项目状态。Agent 先读取真实画布，只处理用户当前要求，不附带维护者私有知识，也不强制套用固定行业流程；本机私有扩展只在 Git 忽略文件实际存在时启用。

Agent 可以：

- 读取画布与选区。
- 创建、更新、删除和连接节点。
- 调用文本、图片、视频和音频生成。
- 按需维护项目约束、已确认常量、待确认问题和下一步。
- 调用 3D 导演台并把预演截图写回画布。

本机 Agent 已可使用 Codex app-server 或 Claude CLI；Codex 对话按画布创建或恢复，并绑定对应工作区。其 token 和工作区只存在用户电脑。

## 3D 导演台

这是目前唯一必须与当前浏览器画布运行态强耦合的 Agent 能力；其他创作能力通过项目状态和通用节点工具工作。

- 完整分镜包最多 60 个结构化镜头，导演台按每批最多 10 个镜头预演和截图。
- 自动恢复角色、道具、坐标、朝向、缩放、相机位置和 FOV。
- Orbit 相机与角色/道具参数手动微调，并同步回当前分镜包。
- 单镜头或批量输出 `1280x720` PNG。
- 截图作为标准图片节点写回画布，并保留镜头 ID 和槽位元数据。
- 桌面和手机响应式布局。

当前使用 Three.js 与轻量占位人物/道具，定位是空间关系和机位预演，不是完整动画制作软件。真实 GLTF/VRM 资产、骨骼动作与时间轴属于后续能力。

## 可选本地资料工具

- 方法资料、完整案例和视频字幕使用不同收件箱。
- 支持文本、Markdown、字幕、HTML、EPUB、PDF，以及通过 Calibre 读取 MOBI/AZW。
- 视频优先提取已有字幕，没有字幕时可选语音转写。
- 知识卡经过提炼与隔离二次审核；不确定内容只保留为候选，不进入正式检索。
- 完整故事只建立结构索引，不压成通用方法卡，也不复制完整原文进数据库。
- 生成索引、审核报告、原始材料和个人来源清单默认被 Git 忽略，公开 Web 构建不携带；本机 Agent 可按配置读取私有报告。

## 素材、记录与同步

- 本地图片、视频、音频和文本素材库。
- 图片与视频生成历史首屏只读取轻量摘要，打开详情时再恢复完整媒体。
- 可选本地媒体目录和自动导出。
- WebDAV 可同步画布、素材、生成记录及关联媒体。

## 当前边界

- 手机适合查看、对话和基础操作；精细连线与大规模编排仍建议使用桌面端。
- 视频播放依赖上游返回格式、CORS 和媒体代理，详见[视频生成与播放](/docs/overview/video-playback)。
- 公开 Agent 不自带维护者知识包，也不会自动拥有完整书籍、课程、视频字幕或实时平台趋势。
- 3D 导演台当前没有真实角色资产、骨骼动画、物理模拟和剪辑时间轴。
