# 快速开始

## 在线体验

打开 [卡藏提示词画布](https://infinite-canvas-jay.vercel.app/canvas)，使用卡藏账号登录。没有账号时前往 [卡藏提示词库](https://prompt-hubs.com) 注册。

登录负责身份、卡藏模型和受保护接口权限。画布项目仍主要保存在当前浏览器，不会仅凭登录自动出现在另一台设备。

## 本地启动

要求 Node.js 20 或更高版本。

```bash
git clone https://github.com/study666-creme/infinite-canvas-jay.git
cd infinite-canvas-jay/web
npm ci
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)。本地版本默认仍连接卡藏登录服务，可通过服务端环境变量 `PROMPT_HUB_API_BASE` 指向兼容的自建身份服务。

## 配置模型

登录后有两种使用方式：

1. 直接选择卡藏账号可用的图片模型。
2. 打开设置，添加自己的 OpenAI 兼容或 Gemini 渠道，并填写 `Base URL`、`API Key` 和模型。

自定义 API Key 保存在当前浏览器。不要在公共设备或不可信浏览器环境中保存生产密钥。

## 数据备份

- 画布项目可导出 JSON。
- 图片、视频和音频 Blob 位于浏览器 IndexedDB。
- 支持的桌面浏览器可以授权本地媒体目录。
- 需要跨设备时可在设置中配置 WebDAV，并手动执行同步。

部署细节见 [部署与自托管](/docs/overview/deployment)。
