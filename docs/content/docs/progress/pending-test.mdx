# 待验证清单

## 3D 导演台

- 自动回归已覆盖 WebGL 非空、手动调整持久化、12 镜头分为 `10 + 2` 两批、第二批只回写两张 `1280x720` 预览图、长工具栏滚轮浏览和 `390px` 手机布局。
- 手机右上角 Agent 按钮首击已能打开 `374px` 宽的可交互面板；仍需在 iOS Safari 与 Android Chrome 真机复测触摸、软键盘和安全区。
- 仍需用户确认真实创作分镜下的机位默认值、操作手感和截图构图是否满足工作流。
- 真实 GLTF/VRM 资产尚未接入，不属于当前版本完成项。

## Agent

- `creativeProjectState` 已增加 `series` AI 短剧与 `short_film` AI 短片形态：短剧故事建议进入分集，短片故事直接进入单片剧本；需用真实模型验证中文自动判断、用户显式改型和活动约束叠加。
- 画布已经通过 `canvas-agent` 接入 Codex：可按画布创建/恢复对话、绑定工作目录、传递图片附件，并在确认后回写工具结果；仍需在真实长任务中验证线程恢复、工作区隔离和附件临时文件清理。
- 项目级 `creativeProjectState` v2 已成为主记录，黑板改为单向镜像；使用真实长项目验证旧项目迁移、状态不反向覆盖、内容改动后版本失效和项目导入导出。
- 动态调度已能从已确认剧本、资产或分镜继续，并跳过无关旧草稿；用活动、第一/第三人称、仙侠/都市和不同完成度输入做真实模型回归。
- 所有实质创作、连线/引用、审核、预览和生成动作均强制确认；验证网页 Agent 与本机 Agent 在关闭“全部确认”后仍不能绕过。
- 在线模型设置已能发送真实最小请求并区分未配置、测试中、成功和失败；仍需用户配置可用 API 完成一次真实活动创作验收。
- 验证网页 Agent 与本机 Codex/Claude 在断线、会话恢复、工具确认和长任务中的一致性。
- 验证生成取消、节点删除和清空画布后不会出现晚到结果复活节点。

## 创作资料库

- 合订 EPUB 可按书脊条目拆成稳定的独立逻辑来源；长书改为全书分层取样和多批次蒸馏，不再把单次五张召回上限误用为每书卡片上限。
- 扫描 PDF 增加文本密度判断和全书代表页 OCR；知识卡强制中文，并区分核心、专项、扩展和校核作用域。
- 使用用户有权处理的真实书摘、完整剧本、长视频和可访问模型验证 OCR、字幕、二次审核、失败重试与增量缓存。
- 抽样检查知识卡是否有证据摘要、适用边界和禁止误用项，候选卡不得进入正式检索。
- `agent-reviewed` 字幕分块已保留 BVID/视频 ID 和 `[HH:MM:SS-HH:MM:SS]` 证据范围，并要求候选提炼与独立审核分离；用当前 60 条课程字幕验证视觉依赖隔离、跨来源去重和独立暂存索引。
- B 站合集只读展开已验证为 60 条、60 个唯一 BVID；可恢复字幕队列仍需完成剩余条目并对 Whisper 同音错字抽样校订后再蒸馏。

## 模型与媒体

- New API 已上线统一模型目录：当前已审核的 4 个文本、7 个图片和 8 个视频模型均返回明确 `modality`、端点、参数类型/枚举/范围/固定值和实时价格；未知模型只进入未分类列表，不自动上架。
- 卡藏文本、图片和视频模型均由远端目录控制上下架、参数和价格；`1 积分 = ¥0.01`，小数积分不得向上取整。默认模型为 `Image2`、`sd2.0` 和 `创作 5.5`，需用真实账号核对报价、扣费、失败退款与任务记录一致。
- 画布在登录、打开选择器、恢复页面焦点和每五分钟定时刷新模型目录；新增模型应在能力资料审核完成后自动出现，不能要求再次修改画布代码。
- 每次 New API 图片、视频和文本提交前会强制刷新当前价格；价格目录不可用时应返回 503，不能回退到旧价继续生成。文本按实际输入/输出 Token 结算，视频分别覆盖按次和按秒结算。
- 画布内置下拉只接受卡藏返回且模态匹配的模型；切换模型后设置面板应只显示该模型允许的比例、分辨率与参考媒体限制，并把所选参数实际提交给卡藏。
- 重点验证 `image2k4k` 只显示 `3:1`、`1:3`、`21:9`、`9:21`、`2:1`、`1:2`、`16:9`、`9:16`，且 2K 为 5.5 积分、4K 为 9 积分。
- 使用真实卡藏账号验证参考图/视频/音频上传、图片与视频生成、文本 Agent 工具续轮、积分刷新、失败退款和生成记录。
- 验证 Gemini 文本/图片与 OpenAI 兼容渠道切换；音频/视频仍应拒绝不支持的 Gemini 调用格式。
- 验证 OpenAI 视频、Seedance 和 xAI/Grok Imagine 任务的生成、下载、刷新恢复和播放。
- 验证远程网关返回真实媒体二进制，失败时不会把 JSON 错误体写入 IndexedDB。
