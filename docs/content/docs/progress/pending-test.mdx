# 待验证清单

## 3D 导演台

- 自动回归已覆盖 WebGL 非空、手动调整持久化、12 镜头分为 `10 + 2` 两批、第二批只回写两张 `1280x720` 预览图、长工具栏滚轮浏览和 `390px` 手机布局。
- 手机右上角 Agent 按钮首击已能打开 `374px` 宽的可交互面板；仍需在 iOS Safari 与 Android Chrome 真机复测触摸、软键盘和安全区。
- 仍需用户确认真实创作分镜下的机位默认值、操作手感和截图构图是否满足工作流。
- 真实 GLTF/VRM 资产尚未接入，不属于当前版本完成项。

## Agent

- 画布已经通过 `canvas-agent` 接入 Codex：可按画布创建/恢复对话、绑定工作目录、传递图片附件，并在确认后回写工具结果；仍需在真实长任务中验证线程恢复、工作区隔离和附件临时文件清理。
- 本机 Agent 的 SSE 短暂断线不再清空消息、当前对话和待确认动作，面板会保留现场并自动重连；仍需在手机网络切换、Agent 重启和长任务中验证恢复行为。
- 可选项目状态已改为顶层主记录，画布节点只做镜像；使用真实长项目验证旧项目迁移、状态不反向覆盖、内容改动后版本失效和项目导入导出。
- 公开网页与本机 Agent 的默认提示均已收敛为通用画布工具协议，不捆绑私有知识或固定行业流程；验证普通节点、生成、导演台和批量任务不会被可选扩展误触发。
- 本机 `canvas-agent` 已增加可选私有上下文：只对相关创作请求召回 `0-5` 张 `verified` 卡，预览接口返回命中词、来源和可信度；需用真实画布对比启用、关闭及无关请求三种情况。
- 所有写入、删除、连线、生成和其他有后果的动作均遵循界面确认规则；验证网页 Agent 与本机 Agent 无法绕过确认。
- 在线模型设置已能发送真实最小请求并区分未配置、测试中、成功和失败；仍需用户配置可用 API 完成一次真实活动创作验收。
- 验证网页 Agent 与本机 Codex/Claude 在断线、会话恢复、工具确认和长任务中的一致性。
- 验证生成取消、节点删除和清空画布后不会出现晚到结果复活节点。

## 可选本地资料工具

- 使用有权处理的测试材料验证 EPUB/PDF/OCR、字幕、隔离二审、失败重试、增量缓存和跨来源去重。
- 抽样检查候选是否包含可追溯证据、适用边界和审核理由；未经用户确认的候选不得进入私有正式索引。
- 验证生成报告、知识索引、原始材料和个人来源清单均留在 Git 忽略目录，公开构建不包含也不自动加载这些产物。
- 使用私有测试库抽样核对检索作用域、排除规则、命中解释和结果质量，不在公开文档记录来源名称、规模或正文。

## 模型与媒体

- 统一模型目录返回已审核模型的 `modality`、端点、参数类型/枚举/范围/固定值和实时价格；未知模型只进入后台未分类列表，不自动上架。
- 卡藏文本、图片和视频模型由实时目录控制上下架、参数和价格；`1 积分 = ¥0.01`，例如 `¥0.001 = 0.1 积分`，小数积分不得向上取整。默认模型为“全能模型2 · 1K”、`sd2.0` 和“全能模型5.5”。
- 画布在登录、打开选择器、恢复页面焦点和每五分钟定时刷新模型目录；新模型在能力资料审核后进入“模型可选项”，但不会自动勾选到已有用户的下拉列表。
- 每次 New API 图片、视频和文本提交前会强制刷新当前价格；价格目录不可用时应返回 503，不能回退到旧价继续生成。文本按实际输入/输出 Token 结算，视频分别覆盖按次和按秒结算。
- 模型选择器只显示与文本、图片、视频或音频能力匹配的条目；未配置的默认渠道不会出现，卡藏模型可由用户单独勾选或隐藏，画布不提供 MJ。
- 切换图片模型后，设置面板只显示目录允许的生成张数、比例、分辨率与参考图数量，并把合法参数实际提交；“全能模型2 · 高质量”支持 `1-4` 张，其余当前图片模型固定 `1` 张。
- 普通用户界面只展示公开名称、能力、参数、积分和人民币价格；真实模型映射、请求阶段和服务日志只在管理员“画布调用”页面可见。
- 使用真实卡藏账号验证参考图/视频/音频上传、图片与视频生成、文本 Agent 工具续轮、积分刷新、失败退款和生成记录。
- 验证 Gemini 文本/图片与 OpenAI 兼容渠道切换；音频/视频仍应拒绝不支持的 Gemini 调用格式。
- 验证 OpenAI 视频、Seedance 和 xAI/Grok Imagine 任务的生成、下载、刷新恢复和播放。
- 验证远程网关返回真实媒体二进制，失败时不会把 JSON 错误体写入 IndexedDB。
