# 待测试

- 画布移动端可用性：手机/窄屏打开画布时，Agent 使用覆盖式面板，聊天输入、工具确认、本地/在线切换可操作；空白画布支持单指拖动、双指缩放；底部工具栏、缩放控件、顶部 Agent 入口和“我的资产”侧栏不应互相遮挡。
- **视频生成与画布播放（Jay fork）**：Seedance 任务成功后节点是否落盘；`/v1/media/fetch` 与 `/tasks/{id}/content` 是否返回真 MP4；播放器重试能否清除坏 IndexedDB；网关重启后旧节点是否仍可播放。参见 [视频播放与故障排查](/docs/overview/video-playback)。
- Codex App 插件文档：新增插件安装教程，区分 AI 自动安装、手动安装和本仓库开发调试安装；同步优化插件 README 与技能说明。
- 配置与用户偏好：每个模型渠道新增 OpenAI / Gemini 调用格式选择，默认 OpenAI，切换默认格式时同步更新默认 Base URL；渠道 Tab 增加紧凑醒目的提醒，说明模型是否可选需要到“模型”Tab 选择。
- Gemini 调用格式：支持拉取 Gemini 模型列表，并用于文本问答、在线 Agent 工具调用、基础生图和图生图请求。
- 音频和视频生成：选择 Gemini 调用格式时先给出不支持提示，仍需使用 OpenAI 兼容渠道。
- 画布 Agent：在线 Agent 与本机 Agent 已增强为全链路创作总监；需验证故事/剧本/分镜/导演/视觉/AI 视频/标题封面/网感任务能自动调用创作知识核心，并输出前做现代审美、平台传播和二次质量审核。
- 视频生成：`grok-imagine-video*` 模型新增 xAI/Grok Imagine 异步接口兼容分支，需验证 zhou API 日志不再出现 0 秒失败后的“异步任务退款”，并确认生成后视频可在画布播放。
- Prompt Hub 画布生图联动：连接图片节点生成时参考图会先压缩，再通过 Prompt Hub 媒体上传换成 `storage://card-images/...` 引用后提交，需用真实 Prompt Hub 账号验证“单独生图节点”和“图片节点连接后图生图”都能成功，且原画布图片不被压缩替换。
- 画布生成记录同步：画布内成功生成的图片会写入 `image_generation_logs`，需验证生成后在 `/image` 生成记录中可见，点击记录能还原提示词、参考图和结果图。
- 画布交互稳定性：按住空格拖动画布后松开空格/鼠标不应继续粘手；删除生成中或生成失败的图片/视频/音频节点后，异步生成结果不应把已删除节点重新补回画布。
- 首屏加载优化：`/image`、`/video` 历史列表改为首屏轻量读取，需用包含大量历史图片/视频的浏览器数据验证列表打开速度、缩略图显示和点击记录后的完整媒体加载。
- 提示词库接口：`/api/prompts` 已改为服务端 revalidate + 远端拉取超时，需验证 GitHub 原始仓库慢或失败时首页/提示词库不会长时间卡住。
