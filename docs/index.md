## 重要边界

- 卡藏登录提供身份和服务权限，不代表画布已经自动云同步。
- 画布、素材和媒体默认保存在当前浏览器，并按卡藏账号隔离；WebDAV 是可选同步。
- 自定义 AI API Key 保存在当前浏览器并由前端直连模型服务。
- 本机 Agent token 能访问本机工作区与画布工具，不能公开或提交到仓库。
